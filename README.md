# GUVI Internship Project

## User Authentication System

This project was developed as part of the GUVI Internship.

## Features

- User Registration
- User Login
- User Profile
- Session Management
- MongoDB Integration
- MySQL Database Support
- Redis Support
- AJAX and jQuery

## Technologies Used

- HTML
- CSS
- JavaScript
- AJAX
- jQuery
- PHP
- MySQL
- MongoDB
- Redis
- Composer

## Project Structure

- assets/
- php/
- vendor/
- composer.json
- composer.lock
- db.php
- guvi.sql
- php.ini

## How to Run

1. Install XAMPP.
2. Start Apache and MySQL.
3. Import `guvi.sql` into MySQL.
4. Install and start MongoDB.
5. Install and start Redis.
6. Place the project inside the `htdocs` folder.
7. Open the project in your browser.
